# EurFelux's Homebrew Tap

Homebrew tap for my apps.

## Usage

```bash
brew tap eurfelux/tap
```

## Casks

| App | Install | Description |
| --- | --- | --- |
| [Lyrisland](https://github.com/EurFelux/Lyrisland) | `brew install --cask lyrisland` | Menu bar lyrics app for Spotify on macOS |
| [Marginalia](https://github.com/EurFelux/marginalia) | `brew install --cask marginalia` | AI-native ePub reader — select text and ask |
